# Databases

## RDS - Relational Database Service

managed service

- MySQL
- postgres
- mariadb

### Amazon Aurora

it's compatible with MySQL and PostgreSQL, provides high performance and availability, and automatically scales alongside your workloads.

gaming applications, media and content management, and real-time analytics.

cloud native unlike RDS where databases run on EC2 instance.

## DynamoDB - NoSQL databases

key value store

managed
NoSQL type of database with tables
scales
flexible data model

## Caching - ElastiCache

in memory storage of frequently used data. sub-millisecond access

redis OSS, valkey, memcached

- reduces load on databases - with caching you might not need very powerful databases.
- improvers performance.

ideal for storing -

- API response
- database query results
- other information that is required repeatedly

ElastiCache - Fully managed

session data management
database query enhancement
gaming leaderboards

high availability
