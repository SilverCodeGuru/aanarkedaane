to toggle between terminal and text editor, use following keybindings
`[
    // Toggle between terminal and editor focus
    {
        "key":     "ctrl+`",
        "command": "workbench.action.terminal.focus"
    },
    {
        "key":     "ctrl+`",
        "command": "workbench.action.focusActiveEditorGroup",
        "when":    "terminalFocus"
    }
]`
